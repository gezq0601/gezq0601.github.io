安装MinGW和Cmake

msvc x86_64版本MinGW

  1. 下载后解压，将mingw64放置到新目录。
  2. 在环境变量PATH中添加mingw64的路径，需要精确到其中的bin子目录（如D:\Cpp\mingw64\bin）

Download CMake

下载后直接安装，有添加到环境变量时勾选即可，否则也需要同上进行配置。

libigl example project

libigl/libigl-example-project: A blank project example showing how to use libigl and cmake. (github.com)

  1. 首先clone该项目
git clone https://github.com/libigl/libigl-example-project.git
  1. 可以按照官网中所述进行操作，但此处不采用
mkdir build
cd build
cmake ..
make
  1. 将该项目用VSCode打开，运行下述命令
cmake -B build
  1. 点击生成该项目，选择打开生成的example.exe即可
